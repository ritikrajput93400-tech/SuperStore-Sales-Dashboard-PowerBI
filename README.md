# SuperStore Sales Dashboard using Power BI

## 📌 Project Overview

This project is a modern neon-themed SuperStore Sales Dashboard built using Power BI.

The dashboard provides interactive visual insights into:
- Sales Performance
- Profit Analysis
- Ship Mode Analysis
- Regional Sales Distribution
- Category & Sub-category Trends
- Monthly Sales & Profit Trends

The main goal of this project is to create a visually attractive and interactive business dashboard for better decision-making and data storytelling.

---

## 🎨 Dashboard Features

- KPI Cards
- Interactive Region Slicer
- Donut Charts
- Monthly Sales Analysis
- Profit Analysis
- Sales by Category
- Sales by Sub-Category
- Regional Sales Map
- Neon UI Dashboard Design

---

## 📌 Key Insights

- West region generated highest sales.
- COD payment mode contributed maximum orders.
- Technology category performed strongly.
- Sales peaked during November and December.
